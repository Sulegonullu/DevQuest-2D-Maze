# DevQuest 2D Maze Game

 Project Description
This project was developed for the DevQuest Game Development Community as a practice assignment.

It is a single-level 2D maze game where the player navigates through obstacles and reaches the exit door to complete the game. When the player reaches the door, a completion message appears indicating successful progress in the DevQuest journey.

Gameplay Features
- 2D character movement
- Jump mechanic
- Rigidbody2D-based physics
- Collision and trigger system
- End-game message upon reaching the door

 Built With
- Unity 6 (LTS)
- C#
- Unity 2D Physics System

 Development Goals
This project was created to practice:
- Player movement scripting
- Physics and collider setup
- Trigger detection
- Scene structure in Unity
- Basic game flow logic
